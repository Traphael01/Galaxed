# GALAXED

A 2D shooter game made entirely in **Python**, using **Tkinter** for the interface and **Pygame** for audio.  
Project developed by **Traphael**, inspired by the retro atmosphere of *Super Metroid* 🛰️  

---

## Description

**Galaxed** is a *space shooter* where you face waves of enemies and increasingly powerful bosses.  
It includes:
- 3 complete levels with progressive waves and unique bosses  
- Main menu with music and buttons: **Play**, **Options**, **Credits**, **Exit**  
- Adjustable **volume** and **resolution** (720p or Fullscreen)  
- Sound effects, GIF animations, and a dynamic background  
- Stamina system, dash ability, and power-ups  

Each boss features its own soundtrack and unique combat mechanics!

---

## Requirements

Make sure you have **Python 3.10+** installed (3.11 or higher recommended).  
Then install all the required libraries with a single command:

```bash
pip install pygame pillow
```
⚠️ Note:

Tkinter and json are part of the Python Standard Library, but some Python builds may not include Tkinter by default.

If the game fails to start with a **“No module named tkinter”** error, reinstall Python from **python.org**
and make sure **“tcl/tk and IDLE”** is checked during setup.


If you don’t have Python installed, download it **here**:
👉 https://www.python.org/downloads/



## How to Run

You can launch Galaxed in two ways:

1. Using the .bat file **(recommended for Windows)**

Simply double-click:

**galaxed.bat**

This automatically runs the game using your Python installation.

for **linux** and **macOS** use:

**galaxed.sh**


2. Using the command line

Open a terminal in the project folder and run:

**python galaxed.py**
